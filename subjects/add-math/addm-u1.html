Functions

Chapter 1: Introduction

  • A function is any given method in mathematics that takes in a value (input) and gives out only ONE value (output)
  • The input of a function is called it's domain (which is the set of all x-values)
  • The output of a function is called it's range (which is the set of all y-values)
  • A function can be plotted on the xy-coordinate system, known as the Cartesian plane
